# FYP

## Vivado

1. Copy the scripts folder into a Vivado project folder
2. From the project folder, run init.tcl

## Web

To run the client (for testing):
```sh
cd booking-system/client
npm start
```

To run the server (for testing):
```sh
cd booking-system/server
npm run dev
```
